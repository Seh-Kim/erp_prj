# 🧾 사원관리 프로그램 (Employee Management System)

> Java Swing을 활용한 데스크탑 기반의 **사원 정보 통합 관리 프로그램**입니다.  
> 사원의 정보 등록, 수정, 조회, 삭제 기능을 포함하며, 관리자와 사원 권한을 구분하여 운영됩니다.

---

## 📌 개발 정보

- **운영체제**: Windows 10  
- **Database**: Oracle  
- **서비스 시스템**: Java SE (실행 파일: Executable JAR)  
- **개발 환경 / 사용 기술**:  
  `Java`, `JDBC`, `Swing`, `Eclipse`, `Golden`  

---

## 🗂️ 프로젝트 개요

- ✅ 사원 등록/조회/수정/삭제 (CRUD)
- ✅ 관리자 전용 기능 (전체 사원 관리)
- ✅ 사원용 기능 (자기 정보 확인 및 일부 수정)
- ✅ Oracle DB 연동 및 예외처리 구현
- ✅ 실행 가능한 `.jar` 파일로 빌드
